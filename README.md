# paper-survey

## Rules
 ○ Submit the paper you intend to read to issues.   
 ○ Summarize the papers you read according to the following format.  

## Format
```md
---
title:  "論文タイトル"
date:   YYYY-MM-DD
categories: ARL EMG...
---

## 1. どんなもの？

## 2. 先行研究と比べてどこがすごいの？

## 3. 技術や手法の"キモ"はどこにある？

## 4. どうやって有効だと検証した？

## 5. 議論はあるか？

## 6. 次に読むべき論文はあるか？

### 論文情報・リンク

- [著者，"タイトル，" ジャーナル名，voluem，no.，ページ，年](論文リンク)
```
